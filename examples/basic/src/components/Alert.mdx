## Confirm modal

```ts
import { confirm, clear } from '@active/react-ui/ConfirmModal';

confirm({
  title: 'confirm',
  content: 'Are you sure to do this?'
})};

confirm({
  content: <span>'Are you sure to do this?'</span>
})};

clear();
```

If you want to have your `Modal` react to different button clicking event when you click on the cancel button or confirm button, you can consider using `<ConfirmModal />`, with its preset APIs `handleOk` and `handleCancel`, you can save your time from constructing the `ModalFooter` and related callbacks that you need to do if you are using `<Modal />`.

```jsx
<Modal rootElement={element}/>

<Modal getRootElement={() => element }/>
```
